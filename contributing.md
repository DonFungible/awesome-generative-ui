# Contributing

Please keep additions relevant to generative UI and useful to practitioners.

1. Add the item to the most appropriate section in `readme.md`.
2. Use the format `- [Name](https://example.com) - Short, factual description or description taken directly from the creators.`
3. Link to the project's primary source and avoid duplicates, referral links, and self-promotional copy.
5. Open a pull request explaining why the resource belongs here.
